Верстка інтернет-магазину

velosklad.org

Один із найбільших проектів. Зверстано 10 сторінок та велика кількість модальних вікон. Сайт мав розміщуватися на OpenCart, тому використав фреймворк Bootstrap 3. Усі іконки в SVG, оптимізовані. Для слайдерів підключав slick.js, для модалок magnific-popup.js.

Інструменти

Photoshop, Illustrator, Sublime Text, Sass/Scss, Gulp, Bootstrap 3
